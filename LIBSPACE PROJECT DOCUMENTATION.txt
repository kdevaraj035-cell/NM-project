1. INTRODUCTION
In modern digital systems, managing information efficiently is very important. Libraries require a
system that can store, update, and manage book records easily. Manual record keeping is difficult
and time-consuming. To solve this problem, a web-based library management backend system
named LibSpace is developed.

LibSpace is a backend application that manages users and books through RESTful APIs. It allows
users to register, log in, and perform book management operations such as adding, updating,
viewing, and deleting books. Each book is associated with a specific user to ensure secure access.
The project mainly demonstrates how authentication and CRUD operations can be implemented
using modern backend technologies like Node.js, Express.js, and MongoDB.
The goal of this project is to build a secure backend system that handles user authentication,
protects data, and manages books effectively.


2. OBJECTIVES OF THE PROJECT

• To develop a backend system for managing library books.
• To implement secure user authentication using JWT tokens.
• To allow users to perform CRUD operations on books.
• To design a scalable backend architecture.
• To demonstrate RESTful API development using Node.js and Express.js.
• To integrate MongoDB database for storing user and book data.
• To maintain a relationship between users and books.

3. PROJECT OVERVIEW

LibSpace is a web-based backend API system designed to manage library operations. The system
provides user authentication and book management functionalities.
Users can create accounts, log in, and manage books associated with their accounts. All operations
are secured using token-based authentication.

Main features include:

6. SYSTEM ARCHITECTURE

LibSpace follows a layered backend architecture with three layers:
API Layer (Routes) – Handles HTTP requests and endpoints.
Business Logic Layer (Controllers) – Contains authentication logic and CRUD operations.
Data Access Layer (Models) – Handles database schema and data storage.
